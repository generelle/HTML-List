Order list attributes

Use "1" for numbered wich is the default or "40" to start the counting .
Use "A" for numbered with uppercase letters.
Use "a" for numbered with lowercase letters. Use "I" for numbered with uppercase roman numbers.
Use "i" for numbered with lowercase roman numbers
